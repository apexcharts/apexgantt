ApexGantt — Draw Tasks on the Timeline

With enableDrawTask: true, drag across an empty part of the timeline to create a task. A preview bar follows the cursor; on release a task is added with the swept date range snapped to the day.

How to interact
  • Press and drag horizontally across an empty row in the timeline to sweep out a date range.
  • Release to create a task; a quick click (no drag) creates nothing.
  • Press Escape mid-drag to cancel.
  • Existing bars stay draggable/resizable, the draw gesture only starts on empty space.
  • Full editing is on too: double-click a cell or bar to edit, right-click for the context menu, drag the bottom handle to set progress, use the toolbar to add/delete, and the bottom row to append a task.
  • Tasks keep their input order (sorting is off), so a task you draw stays where you put it.
