ApexGantt — Sorting & Filtering

Hierarchy-preserving sort (siblings reorder within their parent) plus a view-only filter that keeps matching tasks and their ancestors.

How to interact
  • Click a column header (Task Name, Start, Duration, Progress) to cycle ascending → descending → none.
  • Shift+click headers to sort by several columns at once (a precedence number appears by each caret).
  • Use the Sort by buttons to drive sorting through the public API.
  • Type in the search box to filter by task name; ancestors of matches stay visible.
  • Show incomplete only filters to tasks with progress < 100%.
