ApexGantt — Multi-Selection Demo

Demonstrates enableSelection: checkboxes, keyboard shortcuts, and the public selection API.

How to interact
  • Click a row — select it (replaces current selection)
  • Ctrl / Cmd + Click — toggle a task in the selection
  • Shift + Click — range-select from the last clicked task
  • Checkbox — toggle individual task selection
  • Select-all checkbox (header) — select / deselect all visible tasks
  • Space on a focused row — toggle selection
  • Ctrl / Cmd + A — select all tasks
  • Escape — clear selection
