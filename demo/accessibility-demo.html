⌨️ Keyboard Navigation

  • Click any task row or press Tab to focus the task list.
  • ↑ / ↓ — move between rows
  • → — expand a collapsed parent / move to first child
  • ← — collapse an expanded parent / move to parent row
  • Home — jump to first row  |  End — jump to last row
  • Space — select / deselect the focused row
  • Enter — open the edit form for the focused row (when task editing is enabled)
  • Escape — release focus from the task list
  • Task bars in the timeline are focusable via Tab; press Enter or Space to open the edit form.
